# Guider

Här kommer praktiska guider för organisationer som vill komma igång med cybersäkerhetsarbete utan att börja med ett stort complianceprojekt.

Guiderna ska vara korta, konkreta och möjliga att använda direkt.

## Planerade guider

- Kom igång med riskhantering
- Gör en enkel gap-analys
- Skapa en första åtgärdsplan
- Bygg ett enkelt tillgångsregister
- Följ upp leverantörer utan att drunkna i enkäter
- Skapa en enkel incidentrutin
- Rapportera cybersäkerhet till ledningen
