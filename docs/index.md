# Cyberpraktikan

Cybersäkerhet behöver inte börja med ett stort program, dyra verktyg eller tjocka policyer.

Börja med tre saker:

1. Vem ansvarar?
2. Vilka risker är viktigast?
3. Vad gör vi först?

Cyberpraktikan hjälper mindre och medelstora organisationer att komma igång med praktiskt cybersäkerhetsarbete, utan att fastna i compliance-teater.
