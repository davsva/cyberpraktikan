Här kommer mallar, checklistor och exempel som hjälper organisationer att dokumentera cybersäkerhetsarbetet på en rimlig nivå.

Tanken är inte att skapa dokument för dokumentens skull.

Tanken är att göra det lättare att hålla ordning på ansvar, risker, åtgärder och uppföljning.

## Planerade mallar

- gap-analys
- riskregister
- åtgärdsplan
- incidentlogg
- tillgångsregister
- leverantörsregister
- behörighetsmatris
- backup- och kontinuitetsöversikt
- ledningsrapport
- årshjul

## Status

Mallar kommer att publiceras stegvis.

Vissa enkla checklistor kan bli fria att använda. Mer sammanhängande mallpaket kan senare komma att erbjudas separat.
